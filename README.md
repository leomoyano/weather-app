
# Weather App

## Demo

https://weather-app-lemon-tau.vercel.app/


## Run Locally

Clone the project

```bash
  git clone https://github.com/leomoyano/weather-app.git
```

Go to the project directory

```bash
  cd weather-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
